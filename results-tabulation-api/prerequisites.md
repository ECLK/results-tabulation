# Prerequisites (Mac)

**Python**

`brew install python`

**Virtual Environment**

`virtualenv venv --python=python3`

`pip3 install virtualenv`

**My SQL**

5.7 or above

`brew install mysql@5.7`

`brew link --force mysql@5.7`

`brew services start mysql@5.7`
